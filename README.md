## 📌 Overview

**Bank Transaction Analysis** is a professional data analytics project focused on analyzing banking transaction data to discover valuable business insights, customer behavior trends, transaction patterns, and suspicious activities.

This project demonstrates the complete workflow of a real-world data analyst:

✔ Data Cleaning
✔ Exploratory Data Analysis (EDA)
✔ Feature Engineering
✔ Data Visualization
✔ Business Insights

---

## 🎯 Objectives

* Analyze customer transaction behavior
* Identify popular transaction channels
* Detect unusual login attempts
* Track monthly transaction trends
* Study customer demographics
* Visualize banking performance data

---

## 📂 Dataset Features

The dataset includes the following columns:

* TransactionID
* AccountID
* TransactionAmount
* TransactionDate
* TransactionType
* Location
* DeviceID
* IP Address
* MerchantID
* Channel
* CustomerAge
* CustomerOccupation
* TransactionDuration
* LoginAttempts
* AccountBalance
* PreviousTransactionDate

## 📊 Key Analysis Performed

### 🔹 Data Cleaning

* Removed duplicates
* Handled missing values
* Renamed columns
* Converted date columns

### 🔹 Exploratory Data Analysis

* Transaction type distribution
* Channel usage analysis
* Location-based trends
* Customer age analysis
* Occupation insights
* Account balance patterns

### 🔹 Fraud / Risk Indicators

* High login attempts
* Large transaction values
* Repeated device activity

## 💡 Key Insights
